JOURNAL ENTRY #5

Ok so for this one I created a div and used javascript to make a new one every second. Due to the recent passing of Takeoff from the Migos, I decided to use the album cover for Culture III sort of as a tribute to Takeoff. In the simplest way possible, I think the code works by using a counter except every time the number increases, the number of divs on the page increases. So if the counter is at 7 there will be 7 divs on the page
